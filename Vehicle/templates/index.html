Vechicle Rest API

Ini merupakan Rest API yang dibuat menggunakan Bahasa Pemograman Python dan Framework Django.

Rest API ini dibuat dengan mengimplementasi authentication. Untuk bisa akses api ini bisa dengan memasukan 'Authorization : token ecde0c57c55c73128e2bdac96d0429182791e183' pada header.

Rest API ini juga mengimplementasi pagination dan filter.
